## 📚 Topics Covered

### 🔤 Basics

`main.dart` is the Entry point — Hello World 
`varandprint.dart` is Variables, data types, and printing 
`multi_line.dart` is Multiline strings 
`raw_string.dart` is Raw strings 
### 🔢 Numbers & Math
`numbers.dart` is Number types (`int`, `double`) 
`round.dart` is Rounding numbers 
`asmd.dart` is Arithmetic: Addition, Subtraction, Multiplication, Division 
`str_to_int.dart` is Parsing strings to integers 

### 🔠 Strings

`name.dart` String variables and interpolation 
`fl.dart` Combining of two strings
`string_comp.dart` String comparison 
`string_proper.dart`  to see of they are `isEmpty`, `isNotEmpty`and get ascii code `codeUnits`
`string_trim.dart` Trimming whitespace 
`string_up_down.dart`to upper case and lower case using `toUpperCase()` and `toLowerCase()` 

### 🧠 Logic & Control Flow
`swap.dart` Swapping two variables 
`factorial.dart` Factorial using loops 
`primeornot.dart` Prime number checker 

### 🖥️ User Input
`user_in_string.dart` Reading string input from the user 
`user_in_int.dart` Reading integer input from the user 
`user_in_test.dart` just made to solve doubts and test

## 🚀 Getting Started

### Prerequisites
- [Dart SDK](https://dart.dev/get-dart) installed on your machine

### Run any exercise
```bash
dart run dart_learning/dart_exercise/<filename>.dart
```

**Example:**
```bash
dart run dart_learning/dart_exercise/asmd.dart
```
